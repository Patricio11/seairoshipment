Metaship Public API

Metaship Support

External-facing API for Metaship platform integrations. Authenticate with clientId/secretKey to obtain a Bearer token, then use it to access bookings, stock exports, sailing schedules, and reference data endpoints.

Authentication

  1. Call POST /public/v2/auth with your clientId and secretKey
  2. Receive a JWT accessToken with a 12-hour expiry
  3. Include the token in all subsequent requests: Authorization: Bearer <token>

Scopes

Your API credentials are assigned scopes that control access:

Scope Description
bookings:read View bookings and container details
bookings:write Create bookings
contacts:read View contacts and transporters
contacts:write Update contact details
shipments:read View products, countries, POI, and consignee profiles
stock:read Read stock export data
stock:write Update pallet weights
sailing-schedules:read Search sailing schedules

Rate Limiting

The authentication endpoint is rate-limited to 5 requests per minute to prevent brute force attacks.

Authenticate

Exchange clientId and secretKey for a JWT access token. The returned token should be included in the Authorization header as Bearer <token> for all subsequent API calls.

Rate limited to 5 requests per minute.

required
application/json

    required

    Your API client ID

  • secretKey
    Type: stringFormat: uuid
    required

    Your API secret key (UUID format)

Responses
  • 429

    Rate limit exceeded (max 5 requests/minute)
